# 📒 Contact Book

A simple Contact Book application built using Python. This project allows users to store and manage contacts through a menu-driven interface.

---

## 🚀 Features

- ➕ Add Contact
- 🔍 Search Contact
- 🗑️ Delete Contact
- 📋 Display All Contacts
- 💻 Menu-Driven Console Interface

---

## 🛠️ Technologies Used

- Python 3

---

## ▶️ How to Run

1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Run:

```bash
python contact_book.py
```
